# Bootstrap Confirm Plugin
Confirm dialog based on Bootstrap Modal

## Usage
```html
<form>
    <button class="btn btn-default" 
      data-toggle="confirm" 
      data-title="Wait!" 
      data-message="Are you sure?"
      data-type="success">
      Confirm Form Submition
    </button>
</form>

<a href="#" 
  data-toggle="confirm" 
  data-title="Wait!" 
  data-message="Are you sure?"
      data-type="danger">
  Confirm link
  </a>
```
